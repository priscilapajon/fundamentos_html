TIPS

Escribir en varias líneas

Para poder escribir en varias líneas de código, lo que haremos es sostener la tecla CTRL + ALT y luego presionar la tecla U o la tecla J


Escribir HTML de manera rápida

Vamos a escribir el nombre de la etiqueta y luego damos un ENTER


COMENTARIOS

Para escribir lineas de comentarios vamos a ir a Editar hasta la opción Alternar comentario de línea o bien verificar el acceso por teclado asignado.


Crear un documento HTML rapidamente

Para poder crear una estructura HTML rápido,presionamos SHIFT + 1 Y damos un Enter o click o sobre la primer opción que nos ofrece VSC
